Simulación & Diseño Generativo:

En este ejercicio se nos propuso trabajar a partir de dos elementos, el ala de un avión y una pieza por defecto que incluye Fusion 360: GE Bracket.

¿Qué es el Diseño Generativo?

El diseño generativo abarca diversas áreas como el arte, el diseño de productos o la arquitectura. Puede contribuir y ayudar a procesos de diseño iterativos, teniendo múltiples salidas de diseño.

Es beneficioso el uso del diseño generativo ya que es la manera rápida de generar posibles diseños, así se logran optimizar y modificar las piezas. El software explora las diferentes posibilidades de combinaciones para una solución.
